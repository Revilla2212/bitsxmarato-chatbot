# bitsxmarato-chatbot

application.py - Main function for flask service
chatbot.py - Library of functions for Flask integration
main_chatbot.py - Local chatbot

Flask server Usage:
1-Initialize virtual environment:
    virtualenv .env
    . .env/bin/activate
    export FLASK_APP=application.py
    flask run
2-Access the chatbot through localhost:5000 

Local bot Usage:
1-Execute the python program (python v3.6 needed)
    ./main_chatbot.py
